Exercice 1:

Create a simple HTML page with the following content: " Hello World!". Use at least one title tag.Selecting a tag from the header tags.

Solution for Exercise 1 :

                        
                            <!DOCTYPE html>
                            <html lang="en">
                            <head>
                                <meta charset="UTF-8">
                                <meta name="viewport" content="width=device-width, initial-scale=1.0">
                                <title>Hello World Page</title>
                            </head>
                            <body>
                                <h1 class="function">Hello World!</h1>
                            </body>
                            </html>

Create two different links (using anchor tags) that redirect to Google and Bing respectively.

Exercice 5:

                        </head>
                        <body>
                            <h2>Liste de Courses</h2>

                            <table border="1">

                                    <tr>
                                        <th>Article</th>
                                        <th>Quantité</th>
                                        <th>Prix</th>

Exercice 6 :

Concevez un formulaire de contact simple comprenant des champs tels que nom, email, sujet et message. Utilisez les éléments de formulaire tels que <form> , <input> , <textarea> , <button>.

Solution for Exercise 6 :

                    
                        <!DOCTYPE html>
                        <html lang="en">
                        <head>
                            <meta charset="UTF-8">
                            <meta name="viewport" content="width=device-width, initial-scale=1.0">
                            <title>Formulaire de Contact</title>
                        </head>
                        <body>

                            <h2>Formulaire de Contact</h2>

                            <form action="process_contact.php" method="post">
                                <label for="nom">Nom :</label>
                                <input type="text" id="nom" name="nom" required>

                                <label for="email">Email :</label>
                                <input type="email" id="email" name="email" required>

                                <label for="sujet">Sujet :</label>
                                <input type="text" id="sujet" name="sujet" required>

                                <label for="message">Message :</label>
                                <textarea id="message" name="message" rows="4" required></textarea>

                                <button type="submit">Envoyer</button>
                            </form>

                        </body>
                        </html>
