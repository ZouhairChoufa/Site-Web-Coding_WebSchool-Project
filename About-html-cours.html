HTML Or "Hypertext Markup Language" dates back to the early days of the World Wide Web. Here's a brief overview of the evolution of HTML over time

Html History

1. **HTML 1.0 (1991):** Tim Berners-Lee, the founder of the World Wide Web, created the first markup language to describe documents. It was a simple version of what would become HTML.

2. **HTML 2.0 (1995):** The first official set of HTML specifications was published as RFC 1866. This version included features such as forms and anchor (link) elements.

3. **HTML 3.0 (1997):** This version was developed by the World Wide Web Consortium (W3C). It introduced new features such as tables, applets, and color attributes.

4. **HTML 4.0 (1997-1998):** This version introduced style tags, Cascading Style Sheets (CSS), frames, client-side scripting (JavaScript), and more. HTML 4.01 was the final version of HTML 4.

5. **XHTML (2000):** XHTML (Extensible Hypertext Markup Language) was introduced as a reform of HTML using XML syntax. While XHTML introduced a stricter and more modular approach, it failed to replace HTML in general adoption.

6. **HTML5 (2008):** HTML5 development began in 2004 by the W3C. HTML5 introduced a plethora of new features, including semantic elements (such as < header >, < footer >, < article > ), APIs for graphic drawing, DOM manipulation, local storage management, and native support for audio and video.

7. **HTML Living Standard (2012 - Present):** Since HTML5, the W3C has adopted a "Living Standard" approach, meaning it is continually updated with new features and corrections.

8. **HTML 5.1 (2016) and HTML 5.2 (2017):** These versions brought improvements and clarifications to HTML5.

HTML continues to evolve to meet the changing needs of the web, and its development is guided by the W3C in collaboration with the developer community. HTML remains the fundamental markup language for creating web pages.

Basic Syntax

HTML " Hypertext Markup Language " is used for structuring content on the web. Here are some basic HTML syntax examples:

1. HTML Document Structure :

    
        <!DOCTYPE html>
            <html lang="en">
            <head>
                <meta charset="UTF-8">
                <meta name="viewport" content="width=device-width, initial-scale=1.0">
                <title>Document Title</title>
            </head>
            <body>
                ... Content goes here 
            </body>
            </html>

→The HTML document structure consists of the following elements:

·<!DOCTYPE html> : Declares the HTML version.
·<html lang="en"> : The root element of an HTML page, with the language attribute set to English.
·<head> : Contains meta-information about the HTML document, like character set and viewport settings.
·<title> : Sets the title of the HTML document.
·<body> : Contains the content of the HTML document.

2. Headings : 


            <h1>This is a Heading 1</h1>
            <h2>This is a Heading 1</h2>

            continue in the same thing ... up to h6

→The HTML document structure consists of the following elements:

·<h1> To <h6> : Defines headings of different levels, where <h1> is the largest and <h6> is the smallest.

3. Paragraph : 


            <p>This is a paragraph.</p>

→The HTML document structure consists of the following elements:

·<p> : Defines a paragraph.

4. Links : 


            <a  href= "https://www.example.com"   target=  "_blank">Visit Example.com</a>

→The HTML document structure consists of the following elements:

·<a  href= "https://www.example.com"   target=  "_blank"> : Creates a hyperlink to another page (in this case, "https://www.example.com") that opens in a new tab/window (target= "_blank").

5. Lists: 


            <ul>
            <li> Item 1 </li>
            <li> Item 2 </li>
            ... you can add how much you want of " li " ...
            </ul>

            <ol>
            <li> Item 1 </li>
            <li> Item 2 </li>
            ... you can add how much you want of " ol " ...
            </ol>

→The HTML document structure consists of the following elements:

·<ul>: Defines an unordered list (bullet points).

·<ol>: Defines an ordered list (numbered points).

·<li>: Defines a list item inside <ul> or <ol>.

6. Images : 


        <img  src= "image.jpg"   alt=  "Description of the image si image pas affiche">

→The HTML document structure consists of the following elements:

·<img  src= "image.jpg"   alt=  "Description of the image si image pas affiche"> : Embeds an image with the specified source (src) and alternative text (alt).

7. Forms : 


        <form  action= "/submit"   method=  "post">
        <label  for= "username"   method=  "post"> Username : </label >
        <input  type= "text" id= "username"   name=  "username" required>

        <label  for= "password"> Password: </label >
        <input  type= "text" id= "password"   name=  "password" required>

        <input  type= "submit"   value=  "Submit">
        </form>

→The HTML document structure consists of the following elements:

·<form  action= "/submit"   method=  "post"> : Creates a form that submits data to the server.

·<label> : Defines a label for an input element.

· <input  type= "text" id= "username"   name=  "username" required> : Creates a text input field with an ID, name, and required attribute.

· <input  type= "submit"   value=  "Submit"> : Creates a submit button.

8. Divisions/Containers:


        <div>
            Content goes here
        </div>

→The HTML document structure consists of the following elements:

·<div> : Defines a division or a container to group and style HTML elements.

9. Comments : 


        <!-- This is a comment -->

→The HTML document structure consists of the following elements:

·<!-- This is a comment --> : Adds comments that are not displayed in the browser, useful for notes or explanations in the code.
